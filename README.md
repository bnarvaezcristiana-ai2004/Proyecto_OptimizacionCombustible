# 🌿 OPTML-VC — Optimización del Consumo de Combustible en Cosechadoras de Caña

**Proyecto:** PRY-ML-001 | Programa Talento Tech — Universidad Libre / Ministerio TIC de Colombia

## Descripción
Modelo XGBoost Regressor que predice el consumo de combustible (L/h) en cosechadoras de caña de azúcar del Valle del Cauca, Colombia, a partir de inclinación del terreno, humedad del suelo, densidad del cultivo y velocidad de avance.

**Resultado:** R² = 0.9273 · RMSE = 0.82 L/h · Ahorro estimado: 23.8% · COP $38.179.423/zafra
